HTML

¿Que es HTML?

HTML es el lenguaje de marcado estándar para crear la estructura y el contenido de las páginas web.

Historia de HTML

HTML fue creado por Tim Berners-Lee en 1989-1991 en el CERN para compartir documentos científicos, evolucionando de una sintaxis básica a un estándar web esencial.

HTML5

HTML5 es la quinta y última versión del estándar HTML, diseñada para estructurar y presentar contenido web de forma semántica, moderna y multimedia sin necesidad de plugins externos
